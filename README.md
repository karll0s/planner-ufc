# 🎓 UFC Planner 

O **UFC Planner** é uma aplicação web leve e funcional desenvolvida para estudantes da **UFC (Universidade Federal do Ceará)** em Crateús. Ele permite a organização de horários de aulas, transporte (ônibus), checklists de tarefas e anotações por disciplina, tudo em um único lugar.

---

## ✨ Funcionalidades

-   **📅 Cronograma Dinâmico:** Visualize suas aulas e horários de ônibus por dia da semana.
-   **✅ Checklist:** Controle suas tarefas e estudos pendentes.
-   **📚 Anotações por Matéria:** Espaço dedicado para lembretes e notas de cada disciplina.
-   **⚙️ Customização Total:** Edite dias, horários, salas e nomes das matérias diretamente pela interface.
-   **📂 Backup Inteligente:** Sistema de exportação e importação em JSON que salva **toda** a sua configuração e notas para que você possa migrar entre dispositivos.
-   **📄 Exportação TXT:** Gere um resumo rápido das suas notas para leitura offline.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando tecnologias web padrão (Vanilla), garantindo rapidez e zero dependências externas:

* **HTML5** (Estrutura)
* **CSS3** (Interface moderna com variáveis e design responsivo)
* **JavaScript** (Manipulação de dados e persistência no `localStorage`)

---

## 🚀 Como usar

1.  Acesse o link do projeto (gerado pelo GitHub Pages).
2.  Clique em **⚙️ Configurações** para adaptar o cronograma à sua grade atual.
3.  Use as áreas de texto para suas anotações diárias (o salvamento é automático no navegador).
4.  **Importante:** Periodicamente, use a função de **Exportar Backup** para garantir que seus dados não sejam perdidos caso limpe o cache do navegador.
