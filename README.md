## 🛠️ O SISTEMA

</div>

O **UFC Planner** é uma ferramenta de alta performance desenvolvida para estudantes da UFC. Focada em quem precisa de rapidez, controle total de notas e um visual moderno que não cansa a vista durante as longas jornadas de estudo.

---

<div align="center">

## 💠 DIFERENCIAIS TÉCNICOS

</div>

* **🕒 CRONOGRAMA CYBORG:** Interface escura com destaque neon para as aulas do dia e rotas de ônibus.
* **✅ QUICK CHECKLIST:** Gerencie tarefas e prazos diretamente na tela inicial com sistema de adição rápida.
* **📈 PONTUAÇÕES & MÉDIAS:** Calculadora integrada com simulador de metas: saiba exatamente quanto precisa tirar para ser aprovado.
* **💾 DATA SAFETY:** Backup completo em JSON. Seus dados ficam com você, salvos localmente no navegador.

---

<div align="center">

## 🚀 COMO USAR

</div>

1.  Faça o **Download** dos arquivos `index.html` e `notas.html`.
2.  Coloque ambos na **mesma pasta** (essencial para a navegação entre páginas).
3.  Abra o `index.html` em qualquer navegador moderno.

> 💡 **Dica:** No Android ou iOS, use a opção "Adicionar à Tela de Início" para transformar o Planner em um Web App (PWA) nativo!
